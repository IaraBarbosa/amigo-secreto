# Descrição do Projeto

O projeto é um Challenge do curso *Praticando Lógica de Programação - ONE+Alura*.

O **Amigo Secreto** permite a inclusão dos nomes dos amigos para que sejam sorteados.


# Tecnologias utilizadas

- JavaScript
- HTML
- CSS
